📝 Descrição Completa

SeuFuturo entrega previsões diárias de horóscopo por signo, com leitura clara para amor, carreira e sorte. Escolhe teu signo, acompanha a previsão do dia e desbloqueia leituras mais completas nos planos Premium e VIP.

Recursos Principais:

  • Previsão diária para todos os 12 signos do zodíaco
  • Leituras personalizadas para amor, carreira e sorte
  • Interface rápida, intuitiva e responsiva
  • Design moderno com tema místico e animações suaves
  • Conta de assinante para gerir acesso Premium e VIP
  • Política de privacidade transparente
  • Termos de uso claros e acessíveis
  • Solicitação de dados LGPD disponível
  • Funciona offline após carregamento inicial
  • PWA (Progressive Web App) instalável

🔒 Política de Privacidade

Última atualização: 17 de maio de 2026

Dados Coletados

Coletamos apenas dados mínimos necessários para funcionamento do serviço:

  • Signo zodiacal e plano escolhido
  • Email e nome quando uma conta é criada
  • Dados técnicos agregados para métricas de uso

Como Usamos os Dados

Usamos os dados para:

  • Personalizar previsões de horóscopo
  • Gerir assinaturas e acesso a conteúdos
  • Melhorar a experiência do usuário
  • Cumprir obrigações legais

Compartilhamento de Dados

Não vendemos dados pessoais. Em produção, provedores de pagamento, hospedagem e email devem ser listados nesta política quando forem integrados.

Armazenamento

Este ambiente usa armazenamento local e banco SQLite para desenvolvimento. Em produção, os dados ficam em infraestrutura segura com controle de acesso e auditoria.

Direitos LGPD

Em conformidade com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você pode solicitar:

  • Acesso aos seus dados pessoais
  • Correção de dados incompletos, inexatos ou desatualizados
  • Eliminação de dados desnecessários ou excessivos
  • Portabilidade dos dados a outro fornecedor de serviço
  • Revogação de consentimento
  • Exclusão completa da conta e dados - Saiba como solicitar

Segurança de Dados

Implementamos medidas técnicas e organizativas adequadas para proteger seus dados pessoais contra acesso não autorizado, perda, alteração ou divulgação indevida.

Crianças e Adolescentes

O app não é direcionado especificamente para crianças menores de 13 anos. Caso descubramos que coletamos dados pessoais de crianças sem verificação do consentimento parental, tomaremos medidas para remover essas informações.

Alterações na Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que revise esta página regularmente para acompanhar eventuais mudanças.
